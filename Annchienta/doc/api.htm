Single-Instance and Multi-Instance Classes

All Single-Instance classes can be accessed using annchienta.getClassName(). import annchienta

device = annchienta.getDevice()
You should never use direct constructors for Single-Instance classes. For Multi-Instance classes, you need the contructor. This is always annchienta.ClassName(). import annchienta

surface = annchienta.Surface( "path/to/surface.png" )

Device

The device is the main class in the Annchienta engine. You can perform some global actions with it.

annchienta.getDevice()

Returns the device.

device.setVideoMode( screenwidth, screenheight, fullscreen )

Sets the current video mode. Fullscreen is by default set to False.

Painter

The painter class is used for basic drawing functions.

annchienta.getPainter()

Returns the painter. After this, you usually want to call painter.reset().

painter.reset()

Resets the painter's attributes: matrix, color...

painter.translate( x, y )

Multiplies the current matrix with a translation matrix.
